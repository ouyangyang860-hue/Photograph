专属相册小程序 —— 完整产品设计文档
基于微信小程序 + 云开发 + Three.js 的3D立体相册

一、项目概述
1.1 产品定位
一款带用户体系的3D沉浸式专属相册小程序。照片以立体球（“记忆星球”）形态呈现，用户可自由拖拽旋转、点击“拉出”查看，并支持照片上传、相册管理和主屏展示设置。

1.2 核心特色
特色	说明
立体球展示	所有照片均匀分布在球面上，形成“记忆星球”
自由拖拽	单指旋转球体、双指缩放，带惯性阻尼
点击拉出	点击照片沿径向放大拉出，沉浸查看
用户系统	微信一键登录，数据隔离
灵活管理	上传、删除、设置主屏展示
实时同步	上传后自动出现在3D球体中
二、技术架构
2.1 技术选型
层级	技术方案	说明
前端框架	微信小程序原生	WXML + WXSS + JS
3D引擎	threejs-miniprogram	Three.js 小程序适配版，体积减少40%
后端服务	微信云开发	云数据库 + 云存储 + 云函数，无需自建服务器
用户认证	微信登录 + OPENID	云开发自动注入用户身份
2.2 云开发资源规划
text
云开发环境
├── 云数据库 (3个集合)
│   ├── users        # 用户信息
│   ├── photos       # 照片元数据
│   └── settings     # 用户个性化设置
├── 云存储
│   └── photos/{user_openid}/{timestamp}.jpg
└── 云函数 (3个)
    ├── login        # 用户登录/注册
    ├── uploadPhoto  # 照片上传处理

│   └── 照片间微弱光线连接

    └── 深空渐变 + 动态粒子

位置	控件	功能

底部（半透明渐变）	照片总数	“共 XX 张”

右侧悬浮	切换布局（🔄）	球体/圆环/螺旋切换

3.3.3 手势交互
手势	交互效果
单指拖拽	球体跟随旋转，松手后惯性减速
双指捏合	缩放视角（拉近/拉远）
单击照片	照片沿径向“拉出”放大 → 进入详情
双击空白	重置视角
长按照片	快捷菜单（设为展示/删除）
3.3.4 照片展示逻辑
默认展示：所有已上传照片均匀分布在球面上

主屏展示（⭐置顶） ：标记为“主屏展示”的照片在球面上略微放大并带有光晕，优先吸引视线

非主屏照片：正常大小分布在球面其他位置

3.4 照片详情 — 沉浸查看
功能：单张照片沉浸式浏览

元素	设计说明
照片主体	居中最大化，圆角 + 轻微阴影
背景	毛玻璃效果（3D场景虚化）
信息区	照片下方显示：上传日期 + 是否主屏展示（⭐）
导航	左右半透明箭头（或滑动切换）
关闭	右上角“✕”或下滑关闭
交互：

左右滑动 → 切换照片，球体同步旋转到对应位置

关闭 → 照片沿原路径缩回球面

切换时伴随淡入淡出过渡

四、数据库设计

javascript
{
  _id: "自动生成",

4.2 photos 集合
javascript
{
  _id: "自动生成",
  _openid: "所属用户OPENID",
  fileID: "云存储文件ID",      // 如 cloud://env-id.xxx.jpg
  cloudPath: "存储路径",

  width: 0,                   // 图片宽度
  height: 0,                  // 图片高度

}

五、核心功能流程
5.1 用户登录流程
text
打开小程序
    ↓
wx.cloud.callFunction('login')  // 云函数自动获取 OPENID[reference:7]
    ↓
查询 users 集合
    ├─ 存在 → 返回用户信息
    └─ 不存在 → 创建新用户（昵称/头像从微信获取）[reference:8]
    ↓
进入首页
5.2 照片上传流程（含3D同步）
text
管理页点击上传
    ↓
wx.chooseImage({ count: 9 })[reference:9]
    ↓
循环上传每张图片
    ├─ wx.cloud.uploadFile 上传到云存储[reference:10]
    ├─ 获取 fileID
    └─ 写入 photos 数据库[reference:11]
    ↓
触发3D场景更新
    ├─ 从数据库重新拉取照片列表
    ├─ 重新计算球面分布
    └─ 新照片淡入出现
    ↓
上传完成提示
5.3 主屏展示设置流程
text
管理页点击 ⭐（或批量选择）
    ↓
更新 photos 集合
    └─ db.collection('photos').doc(id).update({ isFeatured: true/false })
    ↓
触发3D场景更新
    ├─ 重新拉取照片列表
    ├─ 主屏照片 → 放大 + 光晕
    └─ 非主屏照片 → 恢复正常大小
    ↓
UI 即时反馈
5.4 3D展厅加载流程
text
进入3D展厅页面
    ↓
查询 photos 集合（按 _openid 过滤）
    ↓
获取照片列表（fileID + isFeatured）
    ↓
初始化 Three.js 场景
    ├─ 创建场景/相机/渲染器
    ├─ 加载所有照片纹理（TextureLoader）
    ├─ 计算球面分布位置
    └─ 主屏照片设置放大 + 光晕
    ↓
启动动画循环（自转 + 交互监听）
